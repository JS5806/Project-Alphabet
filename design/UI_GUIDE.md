# Atomic Design System Guide

- **Atoms**: Buttons, Inputs, Icons (Lucide-React).
- **Molecules**: Todo Items, Search Bar.
- **Organisms**: Todo List, Sidebar Management.
- **Templates**: Dashboard Layout.

## Color Palette
- Primary: #2563eb (Blue 600)
- Success: #22c55e (Green 500)
- Danger: #ef4444 (Red 500)
- Background: #f3f4f6 (Gray 100)
