# UI/UX Design System Guide - Phase 2

## 1. Color Palette
- Primary Background: `#000000` (Pure Black)
- Primary Text: `#FFFFFF` (Pure White)
- Accent/Meta: `rgba(255, 255, 255, 0.4)` (White with 40% Opacity)

## 2. Typography
- Font: Pretendard (Sans-serif)
- Rationale: High legibility in low-light environments.
- Letter Spacing: `-0.05em` for large headers, `0.4em` for labels.

## 3. WCAG Compliance
- Contrast Ratio: 21:1 (Exceeds WCAG 2.1 AAA Standard)
- Line Spacing: Optimized for instant recognition.

## 4. Spacing System
- Base unit: 4px grid.
- Layout uses `Flex` and `Grid` for perfect centering regardless of resolution.
