# UI Design Specification

## Visual Theme
- **Mode**: Dark Mode
- **Background Color**: #282c34 (Abyss Blue)
- **Text Color**: #61dafb (Electric Blue)
- **Font**: Monospace (Consolas, Courier, or D2Coding) to prevent character jumping during time changes.

## Layout
- **Window Size**: 400x150 (Resizable)
- **Alignment**: Center-aligned Label
- **Padding**: 20px internal padding
