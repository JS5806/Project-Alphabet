# AI Powered Lunch Voting App 🍱

AI DevOps 팀이 자동으로 생성하고 배포한 사내 점심 투표 시스템입니다.

## 📋 프로젝트 소개
이 프로젝트는 AI 기반의 DevOps 파이프라인을 통해 구축되었으며, 팀원들이 오늘의 점심 메뉴를 간편하게 투표할 수 있도록 돕습니다.

## 🚀 설치 및 실행 방법

이 프로젝트를 로컬 환경에서 실행하려면 아래 명령어를 터미널에 입력하세요.

```bash
npm install && npm start
```

## 🛠️ 기술 스택
- React
- Node.js
- AI Automated Deployment
