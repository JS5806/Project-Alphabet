# 🕒 심플 파이썬 디지털 시계 (Simple Tkinter Digital Clock)

Python의 `tkinter` 라이브러리를 사용하여 제작된 미니멀한 디자인의 디지털 시계 프로그램입니다. 누구나 쉽고 빠르게 현재 시간을 확인할 수 있도록 가독성과 편의성에 집중하여 개발되었습니다.

---

## 1. 프로젝트 개요 (Overview)

이 프로젝트는 사용자의 요청에 따라 **"검은 배경에 흰색 글씨"**라는 고대비 디자인 원칙을 적용하여 제작되었습니다. 불필요한 기능을 배제하고 시계 본연의 기능인 '정확한 시간 표시'에 충실하며, 저사양 환경에서도 부드럽게 작동하도록 최적화되었습니다.

- **개발 언어:** Python 3.x
- **사용 라이브러리:** tkinter (Standard Library)
- **주요 목적:** 실시간 시간 모니터링 및 데스크톱 위젯 활용

---

## 2. 구현 기능 (Features)

- **실시간 시간 업데이트:** 1,000ms(1초) 주기로 현재 시간을 동기화하여 초 단위까지 정확하게 표시합니다.
- **고대비 UI/UX 디자인:** 시인성을 극대화하기 위해 검은색(#000000) 배경과 흰색(#FFFFFF) 텍스트를 사용했습니다.
- **자동 레이아웃 구성:** 창 크기에 맞춰 텍스트가 중앙에 배치되도록 설계되었습니다.
- **최적화된 로직:** 메모리 누수 방지 및 CPU 점유율 최소화를 위한 효율적인 `after()` 루프를 사용합니다.

---

## 3. 🚀 초간단 실행 가이드 (Quick Start)

파이썬 설치 여부나 복잡한 터미널 명령어를 걱정하지 마세요! 가장 간편한 실행 방법을 제공합니다.

### **"복잡한 명령어 필요 없음!"**

1.  프로젝트 폴더 안에 있는 **'One-Click-Start.bat'** 파일을 찾아 마우스로 더블 클릭하세요.
2.  **자동으로 필요한 프로그램 환경이 설정되고 시계가 즉시 실행됩니다.**
3.  이제 바탕화면 어디든 원하는 곳에 배치하여 사용하세요.
