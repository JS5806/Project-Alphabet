# 📝 심플 투두 리스트 (Simple Todo List)

## 1. 프로젝트 개요 (Overview)
이 프로젝트는 사용자의 일상적인 업무와 일정을 효율적으로 관리할 수 있도록 돕는 **직관적인 투두 리스트(Todo List)** 애플리케이션입니다.
복잡한 기능은 덜어내고, 꼭 필요한 기능만 담아 누구나 쉽고 빠르게 할 일을 기록하고 관리할 수 있습니다.

## 2. 구현 기능 (Features)
본 애플리케이션에 포함된 주요 기능은 다음과 같습니다:

*   **✅ 할 일 추가 (Add Task):** 간편한 입력을 통해 새로운 할 일을 리스트에 즉시 등록합니다.
*   **🗑️ 할 일 삭제 (Delete Task):** 완료했거나 취소된 항목을 리스트에서 제거합니다.
*   **☑️ 완료 체크 (Check Completion):** 할 일의 완료 상태를 토글(Toggle)하여 진행 상황을 파악합니다.
*   **💾 데이터 자동 저장:** 프로그램을 종료해도 작성한 리스트가 사라지지 않고 유지됩니다.
*   **🎨 직관적인 UI:** 사용성을 최우선으로 고려한 깔끔한 디자인을 제공합니다.

## 3. 🚀 초간단 실행 가이드 (Quick Start)

**"복잡한 명령어 입력? 환경 설정? 전부 필요 없습니다!"** 🙅‍♂️

컴퓨터를 잘 모르셔도 괜찮습니다. 아래 방법으로 단 **1초** 만에 실행하세요.

1.  프로젝트 폴더를 엽니다.
2.  폴더 안에 있는 **`One-Click-Start.bat`** 파일을 찾아 **더블 클릭**하세요.
3.  **자동으로 필요한 프로그램이 설치되고, 앱이 바로 실행됩니다.** 🎉

> **참고:** 최초 실행 시에만 필요한 라이브러리를 다운로드하느라 시간이 조금 걸릴 수 있으며, 이후부터는 즉시 실행됩니다.
