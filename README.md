# AI Powered Lunch Voting App

> **AI DevOps 팀이 자동으로 생성하고 배포한 사내 점심 투표 시스템입니다.**

이 프로젝트는 팀원들의 점심 메뉴 결정을 돕기 위해 인공지능 기반으로 제안 및 투표 기능을 제공하는 웹 애플리케이션입니다.

## 🚀 시작하기 (Getting Started)

아래 명령어를 통해 의존성을 설치하고 로컬 개발 서버를 실행할 수 있습니다.

```bash
npm install && npm start
```

## 🛠 기능 (Features)

- AI 기반 메뉴 추천
- 실시간 투표 시스템
- 자동 배포 파이프라인 적용
