# 💄 화장품 유통기한 관리 시스템 (Cosmetic Expiration Manager)

이 프로젝트는 소중한 피부를 위해 화장품의 유통기한과 개봉 후 사용 기간을 체계적으로 관리해주는 스마트 관리 도구입니다.

## 1. 프로젝트 개요 (Project Overview)

많은 화장품을 사용하다 보면 유통기한을 놓치거나, 개봉한 지 너무 오래된 제품을 계속 사용하는 경우가 많습니다. 본 프로젝트는 이러한 문제를 해결하기 위해 다음과 같은 목적을 가지고 개발되었습니다.

- **스마트한 정렬**: 유통기한이 1년 미만으로 남은 제품을 기한이 짧은 순서대로 한눈에 파악할 수 있습니다.
- **위생적인 관리**: 개봉 후 사용 기간이 1년이 지난 제품을 별도로 분류하여 피부 트러블을 예방합니다.
- **간편한 기록**: 화장품의 이름, 유통기한, 개봉일 등을 누구나 쉽게 입력할 수 있는 친숙한 인터페이스를 제공합니다.

---

## 2. 구현된 기능 (Implemented Features)

프로젝트는 총 4단계의 과정을 거쳐 완성되었습니다.

### Phase 1. 요구사항 분석 및 DB 스키마 설계
- 화장품 정보(제품명, 브랜드, 카테고리 등)와 기한 정보(제조일, 유통기한, 개봉일)를 저장하기 위한 데이터베이스 구조를 설계하였습니다.

### Phase 2. 백엔드 API 및 비즈니스 로직 개발
- **유통기한 필터링**: 현재 날짜를 기준으로 유통기한이 365일 미만인 제품을 추출하고 정렬하는 알고리즘을 구현했습니다.
- **사용 기간 계산**: '개봉일'로부터 오늘까지의 날짜를 계산하여 1년이 경과한 제품을 식별하는 로직을 개발했습니다.
- **데이터 입출력 API**: 화장품 정보를 등록, 수정, 삭제할 수 있는 API를 구축했습니다.

### Phase 3. UX/UI 디자인 및 프론트엔드 구현
- 사용자가 한눈에 재고 현황을 파악할 수 있는 대시보드 형태의 인터페이스를 제작했습니다.
- 기한이 임박한 제품은 **주의(주황색)**, 사용 기간이 초과된 제품은 **경고(빨간색)**로 표시하여 시각적 직관성을 높였습니다.

### Phase 4. 통합 테스트 및 최종 안정화
- 입력 데이터의 유효성을 검사하고, 다양한 기기 환경에서 화면이 깨지지 않도록 최적화 작업을 완료했습니다.

## 3. 실행 가이드 (How to Run)

컴퓨터 전공자가 아니더라도 차근차근 따라 하시면 프로그램을 실행할 수 있습니다. 본 가이드는 **Node.js** 환경을 기준으로 설명합니다.

### Step 1: 프로그램 설치 환경 준비
1. [Node.js 공식 홈페이지](https://nodejs.org/ko/)에 접속합니다.
2. **LTS 버전**(안정적인 버전)을 다운로드하여 설치합니다.
3. 설치가 완료되면 컴퓨터를 재부팅해 주세요.

### Step 2: 프로젝트 소스 코드 내려받기
1. 이 저장소의 코드를 다운로드(Zip 파일)하거나, `git clone` 명령어를 사용하여 컴퓨터에 저장합니다.
2. 폴더를 열고 주소창에 `cmd`를 입력하거나 터미널(Terminal) 프로그램을 실행합니다.

### Step 3: 필수 라이브러리 설치
터미널 창에 아래 명령어를 입력하고 엔터(Enter)를 누르세요. 프로그램 구동에 필요한 부품들을 자동으로 설치합니다.
bash
npm install


### Step 4: 프로그램 실행
설치가 모두 끝났다면 아래 명령어를 입력하세요.
bash
npm start

또는 개발 환경인 경우:
bash
npm run dev


### Step 5: 화면 확인하기
1. 명령어를 실행하면 `http://localhost:3000` (혹은 터미널에 표시된 주소)이 나타납니다.
2. 인터넷 브라우저(크롬, 엣지 등)를 열고 해당 주소를 입력하면 화장품 관리 시스템이 나타납니다!
